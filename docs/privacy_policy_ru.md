# Политика конфиденциальности AnimaQuiz

**Дата вступления в силу:** 2026-06-28

## 1. Сбор данных

AnimaQuiz **не собирает, не передаёт и не хранит** личные данные пользователей на серверах. Все данные (прогресс, настройки, статистика) хранятся исключительно в локальной базе данных устройства.

## 2. Данные на устройстве

Следующие данные сохраняются локально (Room/SQLite):

| Данные | Цель |
|--------|------|
| Имя игрока (введённое вручную) | Отображение в профиле |
| Игровой прогресс (уровень, монеты, XP) | Сохранение прогрессии |
| Статистика квизов | Отображение в разделе «Статистика» |
| Настройки (язык, тема, звук) | Пользовательские предпочтения |
| Серия дней, достижения | Игровая механика |

Данные **никуда не передаются** за пределы устройства.

## 3. Разрешения

| Разрешение | Причина |
|------------|---------|
| `POST_NOTIFICATIONS` | Локальные напоминания о серии дней |
| `READ_EXTERNAL_STORAGE` / `WRITE_EXTERNAL_STORAGE` | Экспорт/импорт резервной копии прогресса |

## 4. Локальные уведомления

Приложение использует WorkManager для отправки **локальных** напоминаний о ежедневном квизе. Уведомления не требуют интернет-соединения и не передают данные на сервер.

## 5. Резервное копирование Android

Android может автоматически резервировать данные приложения в Google Drive (если включено в настройках устройства). Конфигурация резервного копирования указана в `res/xml/backup_rules.xml`. Чувствительные настройки из резервных копий исключены.

## 6. Сторонний контент

Часть вопросов на английском языке взята из **Open Trivia Database** (opentdb.com, лицензия CC BY-SA 4.0). Атрибуция указана в разделе «Настройки → Источники вопросов».

## 7. Дети

Приложение не предназначено для детей до 13 лет и не собирает данные детей.

## 8. Изменения политики

В случае существенных изменений политики конфиденциальности номер версии приложения будет обновлён.

## 9. Контакт

По вопросам, связанным с конфиденциальностью: создайте Issue в репозитории проекта.
